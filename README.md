# Generador_contrasenia_rx
generador de contraseñas con react

Intervienen los siguientes:
* Tamaño de contraseña
* Incluir letras mayúsculas
* Incluir letras minusculas
* Incluir numeros
* Incluir simbolos

Iniciar el proyecto
 - npm run dev
